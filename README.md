# tcha-mn.com (live)

This repository hosts the **live** deployment for <https://tcha-mn.com> via the `gh-pages` branch.

## How this repo is used

- **`main`**: lightweight placeholder branch with this README only.
- **`gh-pages`**: published site output (HTML, CSS, JS, assets). This is the source GitHub Pages uses for <https://tcha-mn.com>.

## Source of truth

All source code, content queries, and build logic live in the preview/source repository:

- <https://github.com/tcha-mn/preview.tcha-mn.com>

Changes should be made there; deployments to this repo happen automatically.
